# 그래프 탐색
## DFS vs BFS 비교
### DFS ([코드](./DFS/DFS%20기본%20구현%20(재귀).py))
> 이동과정에서 여러가지 제약이 있을 경우 유리
  - 장점 
    - 구현이 간단(재귀)
    - 별도의 공간 필요하지 않음
  - 단점
    - BFS보다 시간 오래걸림
    - 최단경로 보장x

### BFS ([코드](./BFS/BFS%20기본%20구현.py))
> 최단 경로 구할때
  - 장점 
    - 최단 경로
  - 단점
    - 경로가 매우 길 경우에는 탐색가지가 급격하게 증가함에 따라 보다 많은 기억공간을 필요로 한다

## 구현

### 공통부
- 방문 기록 저장 (1차원 리스트)

### DFS

- 스택 or <u>재귀함수</u>로 구현

### BFS

- 큐 이용
